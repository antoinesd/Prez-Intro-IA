== AlphaFold : résoudre un problème vieux de 50 ans

[.key-insight]
*Le problème* : les protéines se replient selon des règles physiques, mais prédire la bonne forme en 3D depuis la simple séquence d'acides aminés était IMPOSSIBLE — même les meilleurs ordinateurs échouaient depuis 50 ans.

[.columns]
--
[.column]
====
*Avant AlphaFold* :

* Structurale par cristallographie aux rayons X = 2-5 ans de travail
* Coûteux, long, sans garantie
* Seulement ~180 000 structures connues pour les millions de protéines existantes
* Les chercheurs étaient bloqués

*La révolution (2021)* :

AlphaFold2 prédit la structure 3D en **quelques secondes** avec une précision comparable à l'expérience !

== Génération de molécules : l'IA crée des molécules inexistantes

[.key-insight]
*Le problème* : concevoir une molécule avec des propriétés spécifiques (solubles, efficaces, pas toxiques) est un puzzle combinatoire gigantesque. L'IA apprend à naviguer cet espace.

[.columns]
--
[.column]
====
**Comment représenter une molécule à l'ordinateur ?**

* **SMILES** : notation textuelle (comme "CC(=O)Oc1ccccc1C(=O)O" = Aspirine)
* **Graphes** : atomes = nœuds, liaisons = connexions
* **Structures 3D** : vraies coordonnées atomiques

**Ce que l'IA apprend** :

L'IA entraînée sur des millions de molécules réelles apprend des patterns : "si tu veux une molécule soluble, ajoute cet groupe chimique"
